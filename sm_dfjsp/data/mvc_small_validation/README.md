# MVC Small Validation Data

This directory is reserved for the small MVC-SM-DFJSP validation instances generated by:

```bash
python scripts/run_mvc_small_exact_validation.py
```

The script writes `mvc_small_01.json`, `mvc_small_02.json`, and `mvc_small_03.json` here before running exact enumeration. The generated instances are intentionally tiny so that UA assignment, OS ordering, and machine selection can be exhaustively enumerated.
